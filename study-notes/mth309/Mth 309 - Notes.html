0x1.1 System and Linear Equations

方程组可以转换为矩阵，并可以通过 elementary row operation 解出方程组

一个线性系统的所有可能解称为这个线性系统的 solution set

若两个线性系统的所有解相同，则两个线性系统可被视为是等价的equivalent 。换句话说，若线性系统A和B等价，则线性系统A的解同时也是线性系统B的解。

解

  • 若一个线性方程组系统有一个或多个解，则这个系统是有解的( consistent )
  • 若一个线性方程组系统无解，则这个系统是无解的( inconsistent )

Elementary Row Operation (初等行变换)

  • Replacement 行替换 - 使用该行与另一行(或缩放后)的和替换本行
  • Interchange 行调换 - 调换矩阵内的任意两行
  • Scaling 行缩放 - 缩放某一行

 

若两个线性系统通过初等行变换得到相同的矩阵，则这两个线性系统可以备视为 行等价 (row equivalent)

若增广矩阵中的两个线性系统是 行等价的 ，则两个系统会得到一样的解

 

 

0x1.2 Row Reduction and Echelon Forms 行简化与行阶梯式

一个矩形矩阵若符合以下属性，则是一个行阶梯式

  1. 所有项为0的行要在矩阵的最下方

    All non zero rows are above any rows of all zeros.

  2. 行的第一项(leading entry)要在上一行的第一项所在列右侧的列上 Each leading entry of a row is in a column to the right of the leading entry of the row above it.

  3. 每列中第一项下方的项皆为0

    All entries in a column below a leading entry are zeros.

 

若行阶梯式符合下方的附加条件，则为最简行阶梯式(Reduced Row Echelon Form)

  1. 非0行的第一项皆为1

    The leading entry in each nonzero row is 1.

  2. 所有列中的leading entry(为1)是这列中的唯一一个项

    Each leading 1 is the only nonzero entry in its column.

 

Leading entry: 每一行中从左数起第一个非0项

 

A pivot position in a matrix A is a location in A that corresponds to a leading 1 in the reduced echelon form of A.

A pivot column is a column of A that contains a pivot position.

 

Existence and Uniqueness Theorem

当且近当一个增广矩阵的最右列不是一个 pivot column 时，该线性系统才 有解

换句话说，当且近当一个行阶梯式的 [0, 0, 0, 0, ..., 0, b] 行中，b 不为 0 是，这个增广矩阵才有解。

 

当一个线性系统有解时，他的解只有以下两种情况的其中一种：

  1. 有一个唯一的解，且没有任何 自由变量 | free variables
  2. 有一个无数个解，且至少有一个 自由变量 | free variables

0x1.5 Solution Sets of Linear Systems 线性系统的解集

假如一个线性系统可以被写成 的形式，其中 是一个 大小的矩阵、0 是 中的零向量，那么这个系统就是 homogeneous (齐次)。

但 这样的系统，总有至少一个解，即当 时。

这种解被称为 trivial solution （零解；平凡解；无效解；显然解）

而问题在于，我们需要去找到一个 nontrivial solution，即当向量 不为零向量，但仍能满足 的解

 

定理

The homogenous equation has a nontrivial solution if and only if the equation has at least one free variable.

当且仅当 至少有一个 free variable 时，它有 nontrivial solution

普通的线性系统的解可能有三种情况

  1. 无解
  2. 有一个解
  3. 有无数个解

因为 homogeneous linear system 至少有一个解(x = 0)，那么如果我们想要它有 nontrivial solution，则这个系统需要有无数个解，即有 free variable
